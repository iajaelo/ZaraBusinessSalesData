# ZaraBusinessSalesData
This dataset contains sales data for products sold by the Zara brand. It includes detailed information such as product ID, product position inside the store, whether the item was on promotion, product category, season, price, description, origin, and sales volume. Each row represents a single product, giving useful insights into how pricing, store placement, seasonality, and product features affect customer buying behavior. This file is useful for business analysis, retail trend study, data visualization, and understanding customer purchasing behavior and which products perform best in different marketing and seasonal conditions.

This context makes the dataset valuable for business intelligence, data analysis, and decision-making in retail environments



## App.py 
An interactive sales analytics dashboard built with Streamlit using Zara clothing sales data set. The dashboard turns raw CSV data into actionable business insights in seconds allowing business analysts, marketers, or store managers to explore sales performance in real time

## App2.py
An interactive sales analytics dashboard built with Streamlit using Zara clothing sales data set. The dashboard displays an interactive analysis of Zara Jackets including Seasonal, Materials Origin & Sales Volume performance
